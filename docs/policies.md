# Policies

ForgePilot policies and decision records.

## Policy Document Format

Policies are documented as Markdown files in the `docs/` directory.

Each policy document:

* States the decision clearly
* Records the date and context
* Documents alternatives considered
* Records the rationale for the chosen approach

## Scope

ForgePilot policies cover:

* Architecture decisions
* Data collection standards
* Hypothesis evaluation criteria
* Workflow design principles
* Agent interaction protocols

Policy documents are the knowledge truth for ForgePilot.
