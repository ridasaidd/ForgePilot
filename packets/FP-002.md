## PacketMetrics Field Reference

| Field | Type | Description |
|---|---|---|
| `date` | `string` | ISO date of packet execution |
| `packet_id` | `string` | Unique packet identifier (e.g. FP-001) |
| `task_type` | `string` | Task classification |
| `packet_template` | `string` | Template used for the packet |
| `implementation_model` | `string` | Model used for implementation |
| `audit_model` | `string` | Model used for audit |
| `result` | `string` | Outcome: SUCCESS, FAILED, BLOCKED, CLARIFICATION_REQUIRED |
| `first_pass_success` | `string` | Whether first pass succeeded (TRUE/FALSE) |
| `fix_attempts` | `number` | Number of fix iterations |
| `escalated` | `string` | Whether escalation occurred (TRUE/FALSE) |
| `escalated_to` | `string` | Escalation target (human/routing) |
| `audit_accepted` | `string` | Whether audit accepted (TRUE/FALSE) |
| `human_accepted` | `string` | Whether human accepted (TRUE/FALSE) |
| `clarification_required` | `string` | Whether clarification was needed (TRUE/FALSE) |
| `clarification_count` | `number` | Number of clarification rounds |
| `clarification_resolved` | `string` | Whether clarifications were resolved (TRUE/FALSE) |
| `root_cause_level` | `string` | Classified root cause level |
| `root_cause_reason` | `string` | Description of root cause |
| `duration_minutes` | `number` | Execution duration in minutes |
| `notes` | `string` | Free-form notes |
